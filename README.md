# Crop and Download Image 🖼️✂️⬇️

A user-friendly web application for cropping and downloading images directly in your browser. Upload an image, select your desired crop area, and download the cropped version instantly.

## ✨ Features

- 📤 **Upload Images**: Support for various image formats (JPEG, PNG, etc.)
- 🔄 **Crop Functionality**: Interactive cropping with Cropper.js library
- 📐 **Aspect Ratios**: Predefined ratios (16:9, 4:3, 1:1, 2:3) and free-form cropping
- 📏 **Custom Dimensions**: Set specific width and height for cropping
- 👀 **Live Preview**: Preview cropped image before downloading
- 💾 **Download**: Save cropped images as PNG files
- 📱 **Responsive Design**: Works on desktop and mobile devices

## 🚀 Live Demo

Check out the live demo: [https://iam269.github.io/Crop-and-Download-Image/](https://iam269.github.io/Crop-and-Download-Image/)

## 🛠️ Technologies Used

- **HTML5**: Structure and markup
- **CSS3**: Styling and responsive design
- **JavaScript**: Interactive functionality
- **Cropper.js**: Image cropping library

## 📖 How to Use

1. **Upload Image**: Click "Choose A Photo" to select an image from your device
2. **Crop**: Use the cropping handles to select your desired area
3. **Set Aspect Ratio** (optional): Choose from predefined ratios or select "Free" for custom cropping
4. **Adjust Dimensions** (optional): Enter specific width and height values
5. **Preview**: Click "Preview" to see the cropped result
6. **Download**: Click "Download" to save the cropped image as a PNG file

## 🏃‍♂️ Running Locally

Since this is a static web application, you can run it locally by:

1. Cloning or downloading the repository
2. Opening `index.html` in your web browser

No additional setup or server required!
